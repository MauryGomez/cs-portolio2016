How its made: Navbar

When first starting this website, I learned how to put together the navbar that is at the top of each page. First I needed to enter the tag ul to make an unordered list. After, I needed to add in the links under the ul tag to redirect the user to different parts of my page. Using the link tag and the href, I entered the links to redirect the users to the other pages on my site, after the ending carrot for the href tag I added the word that will display to the user for them to interact with. Decorating the Navbar

To decorating or add aesthetics to the navbar you will either need to make a style tag or what I did, is using css in an external page to style the navbar. Fisrt, I used the text-align:center; to center the navbar options. Then I used background-color:black; to color the back ground of the navbar black. After that I started a new section relating to the links in the navbar, using the display css property to show the navbar link in a certain way, in this case, I wanted the links to be solid blocks(so you would use display:solid block;), and in a horizontal line (using display:inline-block;). For the border and padding property, I used it to change the size of my border and padding around each text in my navbar.
